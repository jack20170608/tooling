### 2.3 系统环境初始化
- [执行setup_environment.sh](common/setup_environment.sh)

这个脚本基本也是AI
生成的,
脚本的功能会创建一些文件夹，并在`/home/<username>/.bashrc`
里面添加环境变量，这里贴一下。

```shell
## 创建运行时文件夹
mkdir -pv /appvol/ilovemyhome/{apps,bin,config,data,install,logs,runtime,tmp,download,workspace}
```

```shell
## The /home/<username>/.bashrc
# AUTO-GENERATED ENV VARS - DO NOT EDIT MANUALLY
# Last updated: 2025-09-21 13:56:22
# Generated by setup_environment.sh v1.0.0
export APP_ROOT=/appvol/ilovemyhome
export APP_BIN=$APP_ROOT/bin
export APP_CONFIG_ROOT=$APP_ROOT/config
export APP_DATA_ROOT=$APP_ROOT/data
export APP_INSTALL_ROOT=$APP_ROOT/runtime
export APP_LOG_ROOT=$APP_ROOT/logs
export APP_RUNTIME_ROOT=$APP_ROOT/runtime
export APP_TEMP_ROOT=$APP_ROOT/tmp
export APP_DOWNLOAD_ROOT=$APP_ROOT/download
export PATH=${PATH}:$APP_ROOT/bin

```

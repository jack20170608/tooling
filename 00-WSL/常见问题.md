## git clone时 warning: remote HEAD refers to nonexistent ref, unable to checkout
```shell
## 然后查看有哪些分支：
git branch -a

## 直接切到可用分支
git checkout main || git checkout master
```

## git command line 没有显示当前分支名称 

```shell
# 清理旧的错误配置
sed -i '/parse_git_branch/d' ~/.bashrc
sed -i '/PS1=.*parse_git_branch/d' ~/.bashrc

# 添加兼容 conda 的 Git 分支显示函数
cat >> ~/.bashrc << 'EOF'

parse_git_branch() {
    git branch 2>/dev/null | grep '*' | sed 's/* \([^ ]*\)/ (\1)/'
}

# 保留 conda 环境，只在 Git 目录显示分支
PS1='\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[33m\]$(parse_git_branch)\[\033[00m\] \$ '
EOF

# 立即生效
source ~/.bashrc

```

# FAQ

## git clone时 warning: remote HEAD refers to nonexistent ref, unable to checkout
```shell
## 然后查看有哪些分支：
git branch -a

## 直接切到可用分支
git checkout main || git checkout master
```

## git command line 没有显示当前分支名称 

```shell
echo "parse_git_branch() { git branch 2> /dev/null | sed -e '/^[^*]/d' -e 's/* \(.*\)/ (\1)/'; }" >> ~/.bashrc
echo "export PS1=\"\[\033[01;32m\]\u@\h\[\033[00m\]:\[\033[01;34m\]\w\[\033[33m\]\$(parse_git_branch)\[\033[00m\] \$ \";" >> ~/.bashrc
source ~/.bashrc

```

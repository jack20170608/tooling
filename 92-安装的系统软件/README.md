# 安装的系统软件

这里的系统软件，是指可以通过包管理器安装的系统软件，跟具体的应用软件不同，系统软件是系统的基础，提供系统的运行环境。我把一些编程语言相关的runtime也算作系统软件。
这个可能有些争议，但是对于我而言，与具体的业务无关的软件，可以作为系统软件。
